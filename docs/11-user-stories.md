# 11. User Stories

* **US-01:** As an authenticated user, I want to create a new subscription log entry with specific pricing parameters so that I can track my accounts in a central database.
* **US-02:** As a dashboard viewer, I want to view my calculated monthly burn rate clearly so that I can make accurate budget decisions.
* **US-03:** As an active user, I want to edit my subscription metrics whenever pricing structures shift so that my dashboard stays valid.
