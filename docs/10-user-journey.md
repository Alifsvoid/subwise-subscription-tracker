# 10. User Journey Map

1.  **Discovery & Onboarding:** The user creates an account on SubWise and lands on a clean, empty dashboard state.
2.  **Data Logging:** The user clicks "Add Subscription", populates a straightforward entry form (e.g., *Netflix, $15.49, Monthly, Next Billing: June 25*), and hits save.
3.  **Visualization:** The system captures the inputs, mutates the dataset, recalculates total metrics, and immediately updates the layout on the client dashboard.
