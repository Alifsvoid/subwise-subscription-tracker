# 01. Project Overview

## 1. Executive Summary
SubWise is a lightweight, responsive web application built to help individual consumers combat subscription fatigue and track recurring expenditures. In an economy heavily dominated by software-as-a-service (SaaS), media streaming networks, cloud services, and premium memberships, users often lose track of their financial commitments. SubWise centralizes these records into a clean dashboard interface.

## 2. Project Goals
* **Centralize Log Entries:** Provide an explicit interface with full CRUD functionality for user tracking profiles.
* **Financial Visibility:** Compute monthly and annual expense burn rates dynamically.
* **Prevent Auto-Billing Traps:** Track renewal timelines with intuitive status indicators.

## 3. Technology Alignment
The system relies on a clean client-server pattern. The frontend leverages semantic HTML5 alongside custom CSS3 Flexbox layouts for dashboard responsiveness. The backend processes calculations and interfaces with a relational dataset matching users to their personal data logs.
