# 13. Functional Requirements

* **FR-01 (Auth Access):** The system must restrict core dashboard access routes to authenticated users with valid session tokens.
* **FR-02 (Data CRUD):** The user interface must support individual Create, Read, Update, and Delete operations on subscription records.
* **FR-03 (Calculation Logic):** The system backend must aggregate all subscription rows tied to a user ID to compute exact monthly and annual expenses.
