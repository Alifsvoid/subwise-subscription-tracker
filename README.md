# SubWise - Subscription Tracker

A comprehensive subscription tracking application designed to help users manage, monitor, and optimize their recurring expenses.

## 📁 Repository Structure

This repository is organized into a monorepo structure containing the following core workspaces:

* **`/frontend`**: The user interface and client-side application. 
  * *Tech Stack:* React, Next.js
* **`/backend`**: The REST API, core business logic, and database management. 
  * *Tech Stack:* Python, FastAPI, SQLAlchemy
* **`/docs`**: Project documentation, system architecture diagrams, and milestone tracking.

## 🚀 Environment Setup

### Frontend Requirements
Navigate to the `frontend/` directory. Dependencies are managed via standard Node package managers.
- Core frameworks listed in `package.json`.

### Backend Requirements
Navigate to the `backend/` directory. Dependencies are managed via Python pip.
- Core frameworks listed in `requirements.txt`.
